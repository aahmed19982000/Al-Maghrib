{% trans "حقوق الملكية الفكرية" %}

{% trans "جميع المحتويات المنشورة على الموقع، بما في ذلك الأخبار، المقالات، التحليلات، الصور، الفيديوهات، الشعارات، والتصميمات، هي ملكية فكرية حصرية لموقع المغرب العربي أو شركائه من وكالات الأنباء، وهي محمية بموجب قوانين الملكية الفكرية وحقوق النشر الدولية." %}

{% trans "لا يُسمح بنسخ أو إعادة إنتاج أو نشر أو توزيع أي جزء من محتوى الموقع دون الحصول على إذن كتابي مسبق من إدارة الموقع، مع وجوب ذكر المصدر بوضوح ووضع رابط مباشر للمادة الأصلية عند الاقتباس المصرح به." %}

{% trans "الاستخدام المقبول" %}

{% trans "عند استخدامك للموقع، فإنك تلتزم بعدم:" %}

  • {% trans "استخدام الموقع بأي طريقة تؤدي إلى إلحاق الضرر به أو تعطيل خوادمه أو شبكاته المتصلة." %}
  • {% trans "نشر أو إرسال أي محتوى يحتوي على برمجيات خبيثة أو فيروسات تهدف إلى إتلاف أو اختراق الموقع." %}
  • {% trans "استخدام أدوات آلية أو برمجيات كشط البيانات (Scraping) لجمع محتويات الموقع دون إذن مسبق." %}
  • {% trans "استخدام الموقع للترويج لإعلانات تجارية غير مصرح بها أو إرسال رسائل غير مرغوب فيها (Spam)." %}

{% trans "إخلاء المسؤولية" %}
